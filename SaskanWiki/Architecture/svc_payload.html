Service Message Payload

A common class called "MsgSequencer", implemented via module "msg_sequencer.py", handle two asyncio tasks:

  • The "read_msg" task, which reads bytes from a stream, pulling in the first 4 bytes to get message size, the message package itself.
  • The "send_msg" task, calculates the size of message, sends that in 4 bytes, followed by the message package.

At present, this class is delivered via the "bow_quiver" repository. Will likely move it into a common repo with the rest of the code in the interest of keeping the code base consolidated and less confusing.
